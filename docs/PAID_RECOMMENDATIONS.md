# 정확도·신뢰도 향상을 위한 유료/선택 도구 (참고)

이 프로젝트는 **Lighthouse, axe, Puppeteer, CrUX(선택), 규칙 기반 등급** 등 무료·저비용 소스를 우선 사용합니다.  
아래는 **예산을 쓸 만한 경우** 검토할 수 있는 상용·서비스형 옵션입니다. (목록만 제공하며, 특정 업체 추천은 아님.)

## 검색·색인·트래픽 맥락

| 영역 | 설명 |
|------|------|
| **Google Search Console 연동** | 색인, 검색 쿼리, 클릭/노출 등 (Per-site, OAuth). “URL만 크롤”로는 알 수 없는 부분 보완. |
| **Bing Webmaster Tools** | 보조 색인·크롤 정보. |
| **상용 SEO 스위트 API** | Ahrefs, Semrush, Moz 등 — 백링크·키워드 난이도·경쟁 페이지 비교. 월 구독/API 과금. |

## 실측·합성 모니터링

| 영역 | 설명 |
|------|------|
| **합성 모니터링 서비스** | Checkly, Pingdom, Datadog Synthetics 등 — 지역·주기·알림 기반 레이턴시/가용성. |
| **RUM(실사용자 모니터링)** | Datadog, New Relic, Cloudflare Web Analytics 등 — CrUX보다 **본인 사이트** 트래픽에 특화. |

## 보안·공격면

| 영역 | 설명 |
|------|------|
| **침투/취약점 스캔** | 자동 DAST, 버그바운티 플랫폼 — 헤더 검사보다 한 단계 깊은 검증. |

## AI·콘텐츠 품질

| 영역 | 설명 |
|------|------|
| **더 강한 LLM / 검증 파이프라인** | 이미 사용 중인 API 상위 모델, 또는 검색 증강(RAG)·출처 강제 JSON 스키마 등 **프로세스** 유료화. |

## 이미 결정 시 추천 순서 (비용 대비)

1. **Search Console OAuth** (사이트 소유자 대상 제품일 때).  
2. **RUM** (실제 이탈·성능 병목이 중요한 서비스).  
3. **SEO 스위트** (키워드·경쟁 분석이 제품 핵심일 때).  

필요하면 Agent 모드에서 Search Console 연동이나 PSI 전용 키 분리 같은 구현 범위를 잡을 수 있습니다.
